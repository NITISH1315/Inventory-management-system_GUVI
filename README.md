# Inventory Management System

## Description
This project is a Java-based Inventory Management System developed using Servlets and Core Java.
The application manages products using in-memory storage (ArrayList) and follows MVC architecture.

## Features
- Add, Update, Delete products
- View inventory list
- Client-side and server-side validation
- Error handling
- Modular and clean code

## Technologies Used
- Java
- Servlet API
- HTML, CSS
- ArrayList (In-memory storage)

## Architecture
- Model: Product.java
- View: index.html
- Controller: InventoryServlet.java

## Note
The system uses ArrayList for simplicity and can be extended to use a database.
